## Свойства

| Свойство                     | Тип                                      | По умолчанию | Описание                       |
| ---------------------------- | ---------------------------------------- | ------------ | ------------------------------ |
| [`children?`](#свойства)     | `React.ReactNode` &#124; `JSX.Element[]` | -            | Ребенок                        |
| [`controlLeft?`](#контролы)  | `React.ReactNode` &#124; `JSX.Element[]` | -            | Слот для контролов слева       |
| [`controlRight?`](#контролы) | `React.ReactNode` &#124; `JSX.Element[]` | -            | Слот для контролов справа      |
| [`size?`](<(#размер)>)       | `'m'` &#124; `'s'`                       | `'m'`        | Размер                         |
| `className?`                 | `string`                                 | -            | Дополнительный CSS-класс       |
| `ref?`                       | `React.Ref<HTMLDivElement>`              | -            | Ссылка на корневой DOM-элемент |

## Ребенок

`children` — содержимое ячейки. Можно передать:

- один React-элемент.
- массив элементов. Компонент автоматически расставит отступы между элементами.

Если элемент — это строка или число, он будет обёрнут в компонент **Text**. Размер элемента будет соответствовать значению в свойстве `size`.

<MdxTabs>

```tsx

// Текст
<HeaderDataCell>Текст</HeaderDataCell>

// Текст и кнопка
<HeaderDataCell>{['Текст', <Button view="clear" iconLeft={IconArrowLeft} onlyIcon size="s" />]}</HeaderDataCell>
```

## Контролы

Чтобы указать один или несколько контролов, используйте свойство `controlRight` и `controlLeft`. При указании нескольких котролов отступы между ними расставятся автоматически.

<MdxTabs>

```tsx
<HeaderDataCell
  controlRight={[
    <Button view="clear" iconLeft={IconUnsort} onlyIcon size="s" />,
    <Button view="clear" iconLeft={IconFunnel} onlyIcon size="s" />,
    <Button view="clear" iconLeft={IconKebab} onlyIcon size="s" />,
  ]}
>
  Текст
</HeaderDataCell>
```

</MdxTabs>

## Размер

Чтобы определить размер компонента, задайте свойство `size`:

<MdxTabs>

```tsx
// size = m
<HeaderDataCell size="m">Текст</HeaderDataCell>

// size = s
<HeaderDataCell size="s">Текст</HeaderDataCell>
```
