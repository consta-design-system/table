## Свойства

| Имя                      | Тип                                                              | По умолчанию | Описание            |
| ------------------------ | ---------------------------------------------------------------- | ------------ | ------------------- |
| [leftSide](#наполнение)  | `ReactNode`                                                      | -            | Наполнение слева    |
| [rightSide](#наполнение) | `ReactNode`                                                      | -            | Наполнение справа   |
| [form](#форма)           | `'default'` \| `'brick'` \| `'brickDefault'` \| `'defaultBrick'` | `'default'`  | Форма компонента    |
| [border](#обводка)       | `string`                                                         | -            | Обводка компонента  |
| className                | `string`                                                         | -            | Класс компонента    |
| ref                      | `React.Ref<HTMLDivElement>`                                      | -            | Ссылка на компонент |

## Наполнение

Компонент Toolbar может быть наполнен слева(`leftSide`) и справа(`rightSide`) . Вы можете передать в Toolbar любые компоненты. Если передать массив компонентов, между ними будет расстояние `var(--space-s)`.

<MdxTabs>

```tsx
<Toolbar
  leftSide={[
    <Button iconLeft={IconArrowLeft} view="ghost" />,
    <Button iconLeft={IconArrowRight} view="ghost" />,
  ]}
  rightSide={<Button label="Применить" />}
/>
```

</MdxTabs>

## Форма

За форму компонента отвечает свойство `form`. Может быть одним из следующих значений:

- `default` - скругленный со всех сторон,
- `brick` - прямоугольный со всех сторон,
- `brickDefault` - прямоугольный сверху скругленный снизу,
- `defaultBrick` - скругленный сверху прямоугольный снизу.

<MdxTabs>

```tsx
<Toolbar form="default" />
<Toolbar form="brick" />
<Toolbar form="brickDefault" />
<Toolbar form="defaultBrick" />
```

</MdxTabs>

## Обводка

За обводку отвечает свойство `border`. Может быть одним из следующих значений:

- `all` - обводка вокруг компонента,
- `top` - обводка сверху компонента,
- `bottom` - обводка снизу компонента.

<MdxInformer>

Компонент не даст обвести только скругленную сторону. К примеру если вы передадите `form="default"` и `border="top"`, то компонент не будет обведен сверху.

</MdxInformer>

<MdxTabs>

```tsx
<Toolbar border="all" form="brick" />
<Toolbar border="top" form="brick" />
<Toolbar border="bottom" form="brick"/>
```
