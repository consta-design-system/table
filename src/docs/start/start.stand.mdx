# Установите зависимости

1. Установите зависимости с помощью команд `yarn add @consta/uikit` или `npm install @consta/uikit`.
2. Установите `peerDependencies` из пакета `@consta/uikit`.

# Подключите тему

Тему обязательно нужно подключать в корне приложения, так, чтобы все компоненты `@consta/uikit` были вложены в компонент [Theme](##LIBS.LIB.STAND/lib:uikit/stand:components-theme-stable).

```tsx
import React from 'react';
import { Theme, presetGpnDefault } from '@consta/uikit/Theme';
import { Button } from '@consta/uikit/Button';

const App = () => (
  <Theme preset={presetGpnDefault}>
    <Button label="Кнопка" />
  </Theme>
);
```
