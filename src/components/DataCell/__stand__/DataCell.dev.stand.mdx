## Свойства

| Свойство     | Тип                                                                | По умолчанию | Описание                                   |
| ------------ | ------------------------------------------------------------------ | ------------ | ------------------------------------------ |
| `children?`  | `React.ReactNode` &#124; `JSX.Element[]`                           | -            | Ребенок                                    |
| `icon?`      | `IconComponent ` &#124; ` IconComponent[]`                         | -            | Иконка                                     |
| `control?`   | `React.ReactNode` &#124; `JSX.Element[]`                           | -            | Слот для контролов                         |
| `indicator?` | `'alert'` &#124; `'warning'`                                       | -            | Индикатор                                  |
| `size?`      | `'m'` &#124; `'s'`                                                 | `'m'`        | Размер                                     |
| `view?`      | `'primary'` &#124; `'alert'` &#124; `'success'` &#124; `'warning'` | -            | Слот для контролов                         |
| `level?`     | `number`                                                           | `0`          | Уровень вложенности                        |
| `truncate?`  | `boolean`                                                          | -            | Заменяет переполненный текст на многоточие |
| `className?` | `string`                                                           | -            | Дополнительный CSS-класс                   |
| `ref?`       | `React.Ref<HTMLDivElement>`                                        | -            | Ссылка на корневой DOM-элемент             |

## Ребенок

`children` — содержимое ячейки. Можно передать:

- один React-элемент.
- массив элементов. Компонент автоматически расставит отступы между элементами.

Если элемент — это строка или число, он будет обёрнут в компонент **Text**. Размер элемента будет соответствовать значению в свойстве `size`.

<MdxTabs>

```tsx
// Число
<DataCell>100</DataCell>

// Текст
<DataCell>Текст</DataCell>

// Текст и бейджик
<DataCell>{['Текст', <Badge size="s" label="Бейдж" />]}</DataCell>
```

</MdxTabs>

## Контролы

Чтобы указать один или несколько контролов, используйте свойство `controls`. При указании нескольких котролов отступы между ними расставятся автоматически.

<MdxTabs>

```tsx

// Один контрол
<DataCell control={<Checkbox size="s" checked />}>Текст</DataCell>
  Текст
</DataCell>

// Два контрола
<DataCell
  control={[
    <Button view="clear" iconLeft={IconArrowDown} onlyIcon size="s" />,
    <Checkbox size="s" checked />,
  ]}
>
  Текст
</DataCell>
```

## Уровень вложенности

С помощью свойства `level` можно добавить отступ для визуального отображения вложенности:

<MdxTabs>

```tsx
import './DataCellExampleLevel.css';

import { IconArrowUp } from '@consta/icons/IconArrowUp';
import { Button } from '@consta/uikit/Button';
import React from 'react';

import { DataCell } from '@consta/table/DataCell';
import { cn } from '##/utils/bem';

const cnDataCellExampleLevel = cn('DataCellExampleLevel');

const ButtonWithButton = () => (
  <Button onlyIcon iconLeft={IconArrowUp} view="clear" size="s" />
);

export const DataCellExampleLevel = () => (
  <>
    {[0, 1, 2, 3].map((level) => (
      <DataCell
        className={cnDataCellExampleLevel('Cell')}
        level={level}
        control={<ButtonWithButton />}
      >
        Уровень вложенности {level}
      </DataCell>
    ))}
  </>
);
```

```css
.ct--DataCellExampleLevel {
  &-Cell {
    border-top: 1px solid var(--color-bg-border);
  }
}
```

<DataCellExampleLevel />

</MdxTabs>
