## Ребенок

Вы можете указать ребенка(`children`) как одним элементом так и передать массив. В случае с массивом компонет расставит отступы между каждым элементом.
Если элемент будет иметь тип `string | number`, то он быдет выводится компонетом `Text` с тем размером который вы укажите.

<MdxTabs>

```tsx

// Текст
<HeaderDataCell>Текст</HeaderDataCell>

// Текст и кнопка
<HeaderDataCell>{['Текст', <Button view="clear" iconLeft={IconArrowLeft} onlyIcon size="s" />]}</HeaderDataCell>
```

</MdxTabs>

## Контролы

Вы можете указать один или несколько контролов используя свойство `controls`.
При указании нескольких котролов отстумы между ними раставятся автоматически.

<MdxTabs>

```tsx
<HeaderDataCell
  control={[
    <Button view="clear" iconLeft={IconUnsort} onlyIcon size="s" />,
    <Button view="clear" iconLeft={IconFunnel} onlyIcon size="s" />,
    <Button view="clear" iconLeft={IconKebab} onlyIcon size="s" />,
  ]}
>
  Текст
</HeaderDataCell>
```

</MdxTabs>

## Размер

Чтобу поменять размер используйте свойство `size`

<MdxTabs>

```tsx
// size = m
<HeaderDataCell size="m">Текст</HeaderDataCell>

// size = s
<HeaderDataCell size="s">Текст</HeaderDataCell>
```

</MdxTabs>

## Свойства

| Свойство     | Тип                                      | По умолчанию | Описание                       |
| ------------ | ---------------------------------------- | ------------ | ------------------------------ |
| `size?`      | `'m'` &#124; `'s'`                       | `'m'`        | Размер                         |
| `control?`   | `React.ReactNode` &#124; `JSX.Element[]` | -            | Слот для контролов             |
| `children?`  | `React.ReactNode` &#124; `JSX.Element[]` | -            | Ребенок                        |
| `className?` | `string`                                 | -            | Дополнительный CSS-класс       |
| `ref?`       | `React.Ref<HTMLDivElement>`              | -            | Ссылка на корневой DOM-элемент |
